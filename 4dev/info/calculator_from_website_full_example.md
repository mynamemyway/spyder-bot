## 5. Итоговое объединённое дерево параметров для Telegram‑бота

Для Telegram‑бота структура должна выглядеть как последовательность вопросов:

1. Тип услуги → 4 калькулятора.
2. Внутри каждого калькулятора:
   - Материал / Толщина / Качество / Срок.
   - Дополнительные опции (обработка, монтаж, доставка…).
3. Расчёт общей стоимости на лету.
4. Вывод краткого резюме и кнопка «Оформить заказ».
